# Simple Interest Calculator

## 📌 Project Description
The Simple Interest Calculator is a basic application that calculates the **simple interest** based on the principal amount, interest rate, and time period provided by the user. It helps users quickly determine the interest earned or payable on a loan or investment.

## 🧮 Formula Used
The calculator uses the standard formula for simple interest:

Simple Interest = (Principal × Rate × Time) / 100

Where:
- **Principal (P)** = Initial amount of money
- **Rate (R)** = Annual interest rate (in percentage)
- **Time (T)** = Time period (in years)

## ⚙️ Features
- Takes user input for principal amount, rate of interest, and time.
- Calculates simple interest instantly.
- Displays the calculated interest.
- Easy to use and beginner-friendly implementation.

## 📥 Inputs Required
The user needs to enter:
- Principal amount
- Interest rate
- Time period

## 📤 Output
The program will display:
- The **Simple Interest** calculated using the formula.
